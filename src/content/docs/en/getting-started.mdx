<h2>What is Xania?</h2>

**Xania** is (just like react) a javascript view library for building reactive UI with JSX syntax. A typical component in xania has props (properties), a view and event handlers.

```jsx
function HelloWorld(props: { name, children }) {
  const { name } = props;
  return (
    <button click={() => log(`hello ${name}!`)}>
      Say hello
    </button>
  );
}
```
