## Create your first xania

Get a new Xania project up and running locally with our `create xania` CLI wizard!

  ```shell {2}
 # create a new project with npm
 npm create xania
 √ name · my-project
 √ title · my-project
 √ description ·
 skip C:\dev\sandbox\my-project
 ? Select how to pull @xania
 > install from npm
   with source code
  ```
